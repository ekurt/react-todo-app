- Data is storing in your browser `Local Storage` (when you're not login)
- Data is storing in your Google account `Firebase` (when you're login)
- Import and Export todos `.json`
- Animate effects
- Sound effects
- Drag-Drop
- Copy the clipboard
- Priority for todos
- Tags for todos
- Sorting
- Auto link detect
